API Carros

Lista de Carros Cadastrar Carro

Bem-vindo à API de exemplo para aprender Flask. Use os links acima para navegar entre a lista de carros e o formulário de cadastro.

Rotas úteis

  • GET /carros — Retorna a lista de carros (JSON).
  • POST /carros — Cadastra um novo carro (JSON).
  • GET /carros/novo — Formulário HTML para cadastrar um carro.

Como testar

Você pode testar a API com o formulário (clicando em Cadastrar Carro) ou usando ferramentas como curl e Postman.
